display

块级元素，行内元素

div 是一个标准的块级元素。一个块级元素会新开始一行并且尽可能撑满容器。

span 是一个标准的行内元素。一个行内元素可以在段落中<span>像这样</span> 包裹一些文字而不会打乱段落的布局。

float

float 属性可以使一个元素脱离正常的文档流，然后被安放到它所在容器的的左端或者右端， 并且其他的文本和行内元素环绕它。

向左或者向右平移，一直平移直到碰到了所处的容器的边框， 或者碰到另外一个浮动的元素。

clear

The clear CSS property specifies whether an element can be next to floating elements that precede it or must be moved down (cleared) below them.

When applied to non-floating blocks, it moves the border edge of the element down until it is below the margin edge of all relevant floats.

position

static

默认值。一个 static 元素表示它不会被“positioned”，一个 position 属性被设置 为其他值的元素表示它会被“positioned”。

relative

relative 表现的和 static 一样，除非你添加了一些额外的属性。

其他的元素的位置则不会发生位置改变，来弥补空隙。

position续

absolute

相对于最近的“positioned”祖先元素定位。如果该元素没有“positioned”祖先元素，那么它 是相对于文档的 body 元素定位。

z-index

HTML元素在一条假想的相对于面向（电脑屏幕的）视窗或者网页的z轴上延伸，它们 依据其自身属性按照优先级顺序占用层叠上下文的空间。

  • DIV #2 - z-index 为 2
  • DIV #3 - z-index 为 4
    • DIV #5 - z-index 为 1，在一个 z-index 为 4 的元素内层叠，所以渲染次序为 4.1
    • DIV #6 - z-index 为 3，在一个 z-index 为 4 的元素内层叠，所以渲染次序为 4.3
    • DIV #4 - z-index 为 6，在一个 z-index 为 4 的元素内层叠，所以渲染次序为 4.6
  • DIV #1 - z-index 为 5

媒体查询

为了实现响应式设计，我们使用百分比宽度来布局。 让内容的呈现根据可以根据设备进行变化，而不需改变内容本身。

.demo-media {
  width: 50%;
}
@media screen and (max-width:600px) {
  .demo-media {
    width: 100%;
  }
}
